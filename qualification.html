① 시험 설정 (Configuración)
  • 시험 기본 정보: 시험명·날짜·소속팀·합격기준·제한시간·문항당 배점 입력
  • 배점 비율: 이론(%) + 실습(%) = 100% 되도록 설정. 실습 만점 기준 별도 입력
  • 실습 항목: (선택) 항목명과 배점 등록 시 항목별 취약점 분석 가능
  • 문제 등록: 직접 입력 또는 엑셀 업로드 (A=문제, B~E=보기, F=정답번호)
  • 응시자 명단: 엑셀 업로드 권장 (이름·사번·공정명KR·공정명ES·입사일 5열)
② 배포 (Distribución)
  • 👥 응시자 명단 탭 — 시험 응시자를 이름·사번·공정명(KR/ES)·입사일로 등록. 엑셀 업로드 가능. 근속년수+일수 자동 계산
  • 📱 모바일 시험 발송 탭 — 문제+명단이 내장된 HTML 파일 생성:
  • ① [📱 모바일 시험 HTML 생성] 클릭 → 시험명_모바일시험.html 다운로드
  • ② 카카오톡 파일 공유 또는 이메일 첨부로 응시자 휴대폰에 전송
  • ③ 응시자: 파일 열기 → 드롭다운에서 사번·이름 선택 → 한 문제씩 풀기 → [📥 제출 및 저장]
  • ④ 저장된 사번.json 파일을 관리자에게 회신
  • ⑤ ③ 채점하기 → 🤖 자동채점 탭에 드래그&드롭 → 자동 채점 완료
  • 📱 모바일 화면: 타이머·진행막대·한 문제씩 표시·5분 경고·시간 종료 자동 제출
  • 🖨️ 종이 답안지 출력 탭 — 시험지+답안지 / 시험지만 / 답안지(OMR)만 선택 → 인쇄
③ 채점하기 (Calificación)
  • 🤖 자동채점: ② 배포 탭에서 생성한 모바일 시험으로 응시 후 저장된 사번.json 파일을 드래그&드롭하면 이론점수 자동 계산
  • 🔧 실습평가 입력: 응시자 카드 클릭 → 항목별 또는 총점 입력 → 저장
    이론 채점 완료 시 자동 합산
  • ✍️ 수동 채점: 명단에서 선택 후 직접 번호 입력
  • 이론/실습 순서 무관 — 둘 다 입력되면 자동 합산
④ 결과 이력 (Historial)
  • 모든 채점 결과 목록 (이름·사번·공정·날짜·이론·실습·총점·합격·등급)
  • 이름/사번 검색, 개별 상세 보기
  • 📊 엑셀 내보내기로 전체 결과 다운로드
  • 응시 이력 관리: 동일 작업자가 여러 번 응시하면 응시일별로 모두 기록
⑤ 통계 분석 (Estadísticas)
  • 이론 약점: 문항별 오답률 막대 그래프 → 문항 클릭 시 ①②③④ 응답 분포 확인
  • 실습 약점: 항목별 평균 득점률 (취약 항목 빨간색)
  • 👤 응시자별: 개인 클릭 → 오답 문항 상세(내 답 vs 정답) + 실습 항목별 취득 점수
  • 공정별: 공정별 이론·실습·총점 평균 및 합격률 비교
  • 근속년수별: 입사일 기준 근속 구간별 성적 분포
⑥ 숙련도·다기능 (Habilidades)
  • 등급 기준 마스터: ⚙️ 탭에서 등급명·최소점수·색상·설명(KR/ES) 설정 후 저장
  • 숙련도 등급 결과: 응시 공정명이 별도 표시되며, 합격 공정만 등급 부여
  • 다기능 현황: 2개 이상 공정 합격자 자동 집계 → 다기능 등급 표시
  • 📊 엑셀 저장: 숙련도등급·다기능현황 시트로 내보내기
⚙️ 공통 기능
💾 배포용 HTML 저장
현재 데이터(문제·명단·결과·설정)를 HTML 파일에 내장하여 저장. USB나 메일로 전달하면 다른 PC에서 열 때 모든 데이터 자동 로드.
🌐 한국어 ↔ 스페인어 전환
우측 상단 🌐 ES 버튼으로 전체 UI 언어 전환. 탭·버튼·테이블 헤더·합격여부 등 전체 변환. 스페인어 현장 직원과 함께 사용 가능.
👥 사용자 관리
관리자 계정으로 로그인 시 사용자 추가/삭제 가능. 기본계정(admin/admin1234)은 삭제 불가.
