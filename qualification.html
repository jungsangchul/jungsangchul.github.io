      품목 사진
      📘 사용자 설명서 / Manual de Usuario

      1. 기본 사용 순서

      단계내용
      품목마스터품목명, 품목코드, 유형, 적정재고, 사진을 등록합니다.
      위치마스터창고 위치를 등록합니다. 예: A/1/2
      입고등록품목, 수량, 위치를 입력하면 LOT 바코드가 자동 생성됩니다.
      출고등록바코드를 스캔/입력하면 품목, 입고위치, LOT 잔량이 자동 표시됩니다.

      2. 바코드 규칙

      형식: ITM-품목코드-YYYYMMDDHHMMSS-SEQ

      예: ITM-SINP-20260523165136-001

      3. 출고등록

      바코드를 입력하고 [적용]을 누르면 품목, 위치, LOT 잔량, 단가가 자동 표시됩니다.

      출고일은 오늘 날짜가 자동 입력됩니다. 출고수량은 사용자가 직접 입력합니다.

      4. 재고조회

      사진이 등록된 품목은 재고조회에서 사진을 클릭하면 크게 볼 수 있습니다.

      5. 데이터 저장/배포

      [데이터 저장]으로 현재 데이터를 저장하고, [데이터 가져오기]로 다른 PC에서 불러올 수 있습니다.
